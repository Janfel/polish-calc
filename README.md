# polish-calc

> A simple calculator for polish notation written in Rust.

Just something I did in my free time.

## Features

- `+`
- `-`
- `*`
- `/`
- `**`
- `neg`
- `abs`
- `sqrt`
- `ln`
- `sin`
- `cos`
- `tan`
- `asin`
- `acos`
- `atan`
- `E`
- `PI`

## Usage

```bash
plc "+ * 2 PI E"
# => 9.00146713563863
```
